# Code of Conduct

## Expected Behavior

Participants in the `matgpr` project are expected to:

- communicate respectfully,
- focus criticism on technical content and reproducibility,
- be clear about assumptions, limitations, and conflicts of interest,
- give appropriate credit for datasets, papers, code, and ideas,
- help maintain a professional environment for materials-informatics research.

## Unacceptable Behavior

Unacceptable behavior includes harassment, discriminatory language, personal
attacks, intimidation, repeated disruptive behavior, or misrepresentation of
scientific results.

## Reporting

Concerns can be reported to the project maintainer through the GitHub
repository or by email when appropriate.

Reports will be reviewed with the goals of protecting contributors, preserving
scientific integrity, and keeping the project productive.

## Scope

This code of conduct applies to project spaces, including issues, pull
requests, discussions, documentation, and community interactions connected to
`matgpr`.
